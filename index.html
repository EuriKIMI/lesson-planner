Cloud setup

Connect Supabase

Required

EduPlan Pro stores lessons in the cloud with Supabase authentication and database tables, so plans stay synced for teachers and students across sessions.

1. Create the tables

Run the schema in supabase/schema.sql inside your Supabase SQL editor.

2. Paste your project keys

Use your project URL and anon key. Lesson data goes to the database, not browser storage.

3. Optional AI upgrade

Set OPENAI_API_KEY before launching Electron for live AI generation. Without it, the app falls back to a structured smart template.
